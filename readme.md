
# Template for web-coding

## Install
```npm install```
___

## Plugins
- autoprefixer
- css-declaration-sorter
- postcss
- sass
- vite
- vite-plugin-sass-glob-import
___

## Usage
```Development Mode```
- npm run dev

```Production Mode```
- npm run duild
___
